# ☀️ Каникулы — семейный планировщик

Веб-приложение для планирования школьных каникул: календарь, заметки, встречи и списки летнего чтения для двух детей. Все данные хранятся локально в браузере — без сервера и регистрации.

## Возможности

- **Календарь каникул** — осенние, зимние, весенние и летние каникулы по рекомендованному графику РФ 2025–2026
- **Заметки** — текстовые записи на выбранный день для каждого ребёнка
- **Встречи** — события с временем или на весь день, список ближайших встреч
- **Летнее чтение** — персональные списки книг с отметками «прочитано», авторами и впечатлениями
- **Обратный отсчёт** — сколько дней осталось до конца лета

## Быстрый старт

### Вариант 1 — просто открыть файл

Дважды кликните на `index.html` в проводнике. Откроется приветственная страница, оттуда — кнопка «Перейти к календарю».

### Вариант 2 — локальный сервер (рекомендуется)

```bash
npm start
```

Откройте в браузере: [http://localhost:3000](http://localhost:3000)
